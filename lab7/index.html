Lab7: Functions

Challenge

The basic tasks are pretty easy. The bonus tasks took me a while to solve. I had no clue on how to solve any of the bonus tasks at the beginning. So I searched for Wes Github to find his Javascript code for this lab. I read all of his code and searched what I didn't understand on Google. I didn't just copy and paste Wes codes. I tried to understand them and came up with my own solutions.Especially for the "upperCaseFirstLetter()" function, I use Wes codes as a kind of guideline, and used a different way to complete the function.

I also added some extra features. I added buttons to allow users to have further actions after putting in their names. If they don't like the new names, regeneration and putting in other names are possible for users. I also decorated the buttons to make them look fancy.

Problems

How to get the buttons work properly was the biggest problem I had. I searched on Google about how to link buttons and JS together so that when I click on that button it can execute some JS codes, and how to use "document.getElementById()" so that I can put the output of my JS code to wherever I want. Those now work well.

Results

The page gives a random name based on the user's input. If the user doesn't like the generated name, he can hit the "Generate Another One" button to get a new random name. If the user wants to use another name, he can then press the "Use Other Names" button to put in other names.
